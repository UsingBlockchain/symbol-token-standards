    • Defined in src/standards/NIP13/services/TransactionService.ts:192

    Given an array of transactions, returns the ones that are eligible to be deposits. A transaction is an eligible deposit when:

    • It has type TRANSFER.
    • The recipient address is the exchange address.
    • Contains tokenId's units.
    • Does not contain mosaics different to tokenId.
    • The message field is not empty.

    Parameters

    • transactions: Transaction[]

      Array of transactions that could be eligible deposits.

    • exchangeAddress: Address

      Exchange central account address.

    • tokenId: MosaicId

      Mosaic identifier to filter.

    Returns TransferTransaction[]
